pytest自動化測試--快速入門

2021-03-16

install

要測試的 function 須在前面加上 test

測試的檔案前面or後面加上 test

pass_test.py

執行 pass_test.py

如果需要更多資訊，可以加上 -v

如果沒有檔案參數， pytest會查看當前目錄和所有子目錄的測試文件（檔案前面or後面有加上test的）並執行

測試的可能結果：

  • PASSED (.) 測試成功
  • FAILED (F) 測試失敗
  • SKIPPED (s) 測試被跳過，可以用 @pytest.mark.skip()或 @pytest.mark.skipif()修飾器告訴pytest來跳過測試
  • xfail (x) 預期測試失敗 @pytest.mark.xfail()
  • XPASS (X) 測試不應該通過
  • ERROR (E) 錯誤
