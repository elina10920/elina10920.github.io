hexo+github 建立個人部落格

事前準備

  • 安裝 node.js
  • 安裝 hexo

初始步驟

建立新文章

部署到遠端 （以 github 為例）

  1. 建立一個新的 respository name為 yourGithubID.github.io
  2. 到 blog資料夾底下安裝
  1. 修改 blog/_config.xml 裡面的 deployment
  1. 部署到遠端
  1. 輸入 github 帳號密碼
  2. 完成！

更換主題

  1. 到 blog資料夾底下
  2. 下載主題
  1. 修改 blog/_config.xml 裡面的 theme
  1. 重新清理及生成
  1. 重新啟動server來確認 local端
  1. 部署到遠端（GitHub）
