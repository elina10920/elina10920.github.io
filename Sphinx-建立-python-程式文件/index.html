Sphinx 建立 python 程式文件

2021-04-07

step1 安裝 sphinx

step2 建立 doc 資料夾來存放 sphinx 文件

step3 建立 sphinx 專案 (會在 doc 資料夾建立 sphinx 文件)

step4 修改 doc/source/conf.py

將註解拿掉

path 路徑加入的是能讓 sphinx 讀取到 python 檔的位置

（在此假設 python 檔案放在 src 資料夾底下）

修改 extensions

sphinx.ext.autodoc : 自動抓取路徑檔案

sphinx.ext.napoleon : docstring 使用 google style

step5 編輯 doc/source/index.rst

index.rst 為預設首頁，加入索引名稱 Example

index.rst

step6 在**doc/source/**目錄底下建立一個 Example.rst 文件

Example.rst

====== 上面是放標題名稱

.. automodule:: Example 會自動抓取在 src 資料夾底下的檔名 Example

:members: 程式檔裡面的 class 名稱或 function 名稱，可以選擇不填

step7 cd 到 doc 目錄底下

就會 build 出 html 檔

預設首頁為 index.html
